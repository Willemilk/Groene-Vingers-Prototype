# Contributing to Groene Vingers

Bedankt voor je interesse in bijdragen aan Groene Vingers!

## Development Setup

1. Fork de repository
2. Clone je fork
3. Maak een branch: `git checkout -b feature/amazing-feature`
4. Maak je changes
5. Commit: `git commit -m 'feat: add amazing feature'`
6. Push: `git push origin feature/amazing-feature`
7. Open een Pull Request

## Commit Convention

We gebruiken conventional commits:

- `feat:` - Nieuwe feature
- `fix:` - Bug fix
- `docs:` - Documentatie
- `style:` - Code formatting
- `refactor:` - Code refactoring
- `test:` - Tests
- `chore:` - Build/tooling

## Code Style

- TypeScript voor alle nieuwe code
- Gebruik Tailwind CSS voor styling
- Volg bestaande code patterns

## Pull Request Process

1. Update README indien nodig
2. Test je changes lokaal
3. Beschrijf je changes duidelijk in de PR
